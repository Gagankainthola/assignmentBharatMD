# Assignment Completed


### dependencies
```sh
npm install
```

### env variables
Create a **.env** file in the root folder and add the following:
```env
PORT=3000
MONGO_URI= your mongo url
REDIS_SECRET="YOUR REDIS SECRET
```

### Start the Server
```sh
npm start
```

Or, for development with **Nodemon**:
```sh
npm run dev
```
